# Enviroments

- Run the following command to create a conda environment

  ```shell
  conda create -n rfm python=3.10
  ```
- Activate the environment

  ```shell
  conda activate rfm
  ```
- Install the necessary python packages

  ```shell
  pip install -r requirements.txt
  ```
- Done.
